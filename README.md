# School Management API

This is a Node.js API built for the Educase assignment. It allows users to add schools and find the ones closest to them based on their location.

## How to Run:
1. Open the folder in VS Code.
2. Run `npm install` in the terminal.
3. Set up your `.env` file with your MySQL details.
4. Run `node index.js`.

## Features:
- **Add School:** Saves school name, address, and coordinates.
- **List Schools:** Shows all schools sorted by distance from you.
